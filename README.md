# 🛍️ Retail Business Performance Analytics

An end-to-end Retail Business Analytics project built using **PostgreSQL, SQL, Python, and Tableau** to analyze sales performance, customer behavior, product performance, and regional trends.

---

## 📌 Project Overview

This project demonstrates a complete data analytics workflow, from raw retail transaction data to actionable business insights.

The workflow includes:

- Data Import & Validation
- Data Cleaning
- Business KPI Analysis
- Customer Analytics
- Sales Analytics
- Tableau Dashboard
- Python Automation

---
